To develop a good web page you need to understand the tools below:

  • HTML
    • Types of Tags
      • Open Tag
      • Closed Tag
      • Comment Tag
    • Default Display
      1. Block Display
      2. Inline Display
  • CSS
    • External CSS
    • Internal CSS
    • Inline CSS
  • JavaScript
    1. It is for functionality
    2. It is client-side scripting
    3. It is an interpreted language
    4. It is not a compiled language

To be a good web developer, follow these steps judiciously:

  1. Always come to class promptly
  2. Always come to class with a laptop
  3. Participate effectively during class sessions
  4. Follow the lecturer closely
    1. Understand the code
    2. Understand the purpose of the code
    3. Do not just type the code
    4. Understand the structure the code is used for
  5. Do not use your device for non-class activities
  6. Ask why a particular code is used
  7. Never copy code you do not understand
  8. Notify the lecturer when in doubt
  9. Ask questions and contribute in class
  10. Do all assignments and submit them on time

Some main web development terms:

CSS
CSS stands for Cascading Style Sheets. It styles web pages to improve appearance and user experience.
